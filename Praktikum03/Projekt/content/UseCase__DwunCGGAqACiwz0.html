Scenario
Steps
1. if Nutzer möchte Note hinzufügen
    1.1. Nutzer wählt Fach aus
    1.2. if noch keine drei Noten eingetragen sind
        1.2.1. neuste Note hinzufügen
         end if
   end if

Scenario2
Steps
1. if Nutzer möchte Note bearbeiten
    1.1. Nutzer wählt Fach aus
    1.2. Nutzer bearbeitet Note
    1.3. Nutzer klickt auf OK
    1.4. Aktualisieren
   end if

Scenario3
Steps
1. if Nutzer möchte Fach löschen
    1.1. Nutzer wählt Fach aus
    1.2. if ausgewähltes Fach ist kein Pflichtfach
        1.2.1. Nutzer bestätigt
        1.2.2. Fach wird gelöscht
        1.2.3. Daten aktualisieren
         end if
   end if

Scenario4
Steps
1. if Nutzer möchte neues Fach anlegen
    1.1. if Fach exisiert noch nicht (Name muss einzigartig sein)
        1.1.1. Fachname und ECTS werden von Nutzer angegeben
        1.1.2. Fach wird hinzugefügt
         end if
   end if

Scenario5
Steps
1. if Nutzer möchte Fach bearbeiten
    1.1. Nutzer wählt Fach aus
    1.2. Nutzer editiert Daten (Namne / ECTS)
    1.3. Nutzer speichert ab
   end if
